Motus Financial Self-Service Installation Guide

Welcome to Motus Financials’ NetSuite payment processor. This guide will show you how to install and configure the bundle quickly and easily. 

Prerequesites

Before beginning this self-install process, please check that the following have been completed:

  1. You have purchased the ‘Motus Financial Payments Bundle | Built For NetSuite’ found here.
  2. You have filled out the merchant account application with Motus Financial.
  3. You have received an email from the Motus Financial team detailing your unique API information for PayTrace. You will need the information in this email for some of the following steps

If you have any questions about the above steps, please use our websites contact page to get in touch with our support team

Install Bundle

Once you receive your PayTrace API information from Motus, the bundle will be ready to install.

In your NetSuite account, navigate to Customization > SuiteBuilder > Search & Install Bundles

On the following page, search for the following bundle: Motus Financial Payments

Select the hyperlink for the Motus Financial Payments which will bring you to the bundle overview page to prepare for an installation. 

Click the Install Button.

This will bring you to the Bundle Preview where you can click on the Install Bundle button.

Click Okay on the confirmation popup to install the bundle into your account. At this point, the Motus Financial Payments bundle will begin installing into your NetSuite environment. You will be brought to a list of your installed bundles that will show the status of the bundle installation. This may take up to 15 minutes to complete. Click refresh at the top of the page to see the updated status.

You have now successfully installed the bundle!

Bundle Configuration

Before you can start to capture payments, there is some configuration that need to be done to connect PayTrace to Netsuite.

In your NetSuite account, in the Global Search type ‘Motus Credentials’ into the search bar and click ‘New Motus Credentials’.

 

On the following record, enter the values into the following fields: 

ENVIRONMENT: production

API BASE URL: https://api.paytrace.com

GRANT TYPE: password

USERNAME : Found in API Information email sent from Motus Financial team

PASSWORD: Found in API Information email sent from Motus Financial team

INTEGRATION ID: Found in API Information email sent from Motus Financial team

All other fields are not necessary to be filled.

 

Click Save on the record.

 

Congratulations, you have successfully completed the self service install!
